## How to Contribute

1. **Fork** the repository  
2. **Create a branch**
   ```bash
   git checkout -b feature/your-feature
   ```
3. Make your changes
   Follow existing code style
   Keep code clean and readable
   
4. Commit clearly
   ```bash
   git commit -m "Add: short description"
   ```
## Reporting Issues
### Please include:
* Clear description
* Steps to reproduce
* Expected behavior
